# Velocity AI Landing Page

## Overview

A modern landing page redesign for an AI workforce management platform. The redesign focuses on improving clarity, visual hierarchy, and conversion while showcasing key product capabilities.

## Objective

Design a clean, conversion-focused landing page that communicates the value of AI-powered workforce management.

## Features

- Modern hero section
- Dashboard preview
- Product feature highlights
- KPI section
- Call-to-action section
- Responsive layout

## Tools Used

- Figma
- Auto Layout
- Components
- Variables
